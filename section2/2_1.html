2.1. Instala Ubuntu 24.04 en una Máquina Virtual (VirtualBox)

Aprenderás a instalar Ubuntu 24.04 en una máquina virtual utilizando VirtualBox, una herramienta clave para quienes no pueden instalar Ubuntu directamente. Esta guía te permitirá seguir el curso sin complicaciones desde cualquier sistema operativo.

🔗 Video Tutorial
En este tutorial, aprenderás a instalar Ubuntu 24.04 en una máquina con Windows o Mac utilizando VirtualBox. Ubuntu sigue siendo el mismo en ambos sistemas; la única diferencia es el instalador de VirtualBox que debes descargar según tu sistema operativo.

Sigue los pasos detallados para completar la instalación correctamente y asegurarte de que tu entorno de Ubuntu funcione sin problemas.

https://youtu.be/6FARNO45SY8


📘 Notas de la clase

Sistemas operativos recomendados

  • ROS 2 Jazzy funciona mejor en Ubuntu 24.04.

  • macOS no es recomendable (tier 3).

  • Windows puede presentar muchos problemas de compatibilidad.

  • Ubuntu ofrece la menor cantidad de inconvenientes, especialmente para principiantes.

Opciones para instalar Ubuntu

  1. Instalación nativa (recomendada): dual boot en tu máquina física.

  2. Máquina virtual: si no puedes modificar tu sistema (por ejemplo, equipos de trabajo).

Descarga de herramientas

  • Ubuntu ISO: busca “Ubuntu 24.04 download” en Google o visita releases.ubuntu.com para elegir la versión Desktop.

  • VirtualBox: descarga desde https://www.virtualbox.org según tu sistema operativo.

Instalación de VirtualBox

  • Ejecuta el instalador y sigue los pasos por defecto (aceptar términos, ubicación, etc.).

  • Asegúrate de instalar los controladores si son necesarios.

Creación de máquina virtual

  • Nombre: algo como ubuntu-24.04-ros2-course

  • Selección de imagen ISO: elige la que descargaste.

  • Sistema: Linux / Ubuntu 64-bit.

  • Desactiva instalación desatendida.

  • Asigna recursos:

    • RAM: entre 4–8 GB (según tu máquina).

    • CPU: la mitad de tus núcleos (por ejemplo, 4 de 8).

    • Disco duro: mínimo 30–40 GB.

Ajustes adicionales (antes de iniciar la VM)

  • En Sistema > Aceleración: desactiva "Enable Nested Paging".

  • En Pantalla > Memoria de video: 128 MB.

  • Desactiva “Enable 3D acceleration”.

Instalación de Ubuntu dentro de VirtualBox

  1. Selecciona "Try or Install Ubuntu".

  2. Configura idioma, teclado, y conexión de red.

  3. Elige instalación interactiva con apps esenciales.

  4. Marca la casilla para instalar software de terceros (drivers, Wi-Fi, etc.).

  5. Método de instalación: Erase disk and install Ubuntu (es seguro en VM).

  6. Configura nombre de usuario, nombre del equipo y contraseña.

  7. Finaliza la instalación y reinicia.

Primeros pasos tras la instalación

  • Abre terminal y ejecuta:

    • sudo apt update

    • sudo apt upgrade

  • Instalación de herramientas para VirtualBox (Guest Additions):

    • Instala paquetes necesarios:

      • build-essential

      • gcc

      • make

      • perl

      • dkms

    • En VirtualBox: ve a Dispositivos > Insertar imagen de CD de las Guest Additions.

    • Abre terminal en la carpeta del CD y ejecuta sudo ./VBoxLinuxAdditions.run.

Finalizando

  • Reinicia la máquina virtual.

  • Configura portapapeles compartido en modo bidireccional.

  • Ubuntu está listo para trabajar con ROS 2 en el resto del curso.
