2.4. Configura tu entorno para ROS 2

Antes de comenzar a trabajar con ROS 2, es necesario configurar correctamente el entorno para que los comandos funcionen en la terminal. En esta clase aprenderás cómo hacerlo de forma manual y cómo automatizar el proceso.

📝 Documentación

https://docs.ros.org/en/jazzy/Tutorials/Beginner-CLI-Tools/Configuring-ROS2-Environment.html

📘 Notas de la clase

  • Aunque ROS 2 ya está instalado, no se puede usar de inmediato desde la terminal sin una configuración previa.

  • Es necesario "sourcear" (cargar) un script llamado setup.bash, ubicado en el directorio de instalación de ROS 2, para que los comandos estén disponibles.

Pasos para configurar ROS 2 en la terminal:

  1. Intentar ejecutar ros2 sin configuración previa genera el error: ros2: command not found.

  2. Verificar que ROS 2 está instalado en:
    /opt/ros/
    Allí habrá una carpeta con el nombre de la distribución instalada, por ejemplo: jazzy.

  3. Dentro de la carpeta jazzy, se encuentra el archivo:
    /opt/ros/jazzy/setup.bash

  4. Para habilitar ROS 2 en la terminal actual, ejecutar:
    source /opt/ros/jazzy/setup.bash

  5. Luego de hacer esto, al ejecutar ros2 ya aparece un mensaje de ayuda y el comando funciona.

Automatizar la carga del entorno en cada terminal:

  1. En lugar de cargar automáticamente ROS 2 al abrir cada terminal, puedes crear una función en tu archivo .bashrc para activarlo solo cuando lo necesites.

    Pasos:

    1. Abrir el archivo .bashrc en tu directorio personal:
      ~/.bashrc

    2. Ir al final del archivo y añadir esta función
      function set-ros() {
          source /opt/ros/jazzy/setup.bash
          env | grep ROS
      }

    3. Guardar y cerrar el archivo.

    4. Cerrar la terminal y abrir una nueva.

    5. Cuando quieras activar ROS 2 en una terminal, solo escribe:
      set-ros

      Esto:

      • Ejecuta el source del entorno de ROS 2.

      • Luego muestra todas las variables de entorno relacionadas con ROS (verificadas con env | grep ROS).

¿Qué es .bashrc?
Es un script que se ejecuta cada vez que se abre una nueva sesión de terminal. Permite personalizar el entorno de trabajo.
