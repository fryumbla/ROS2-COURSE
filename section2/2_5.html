2.5. Ejecuta tu primer programa de ROS 2

En esta clase ejecutaremos dos programas de ejemplo incluidos en ROS 2 para verificar que el entorno esté correctamente instalado: un publicador ("talker") y un suscriptor ("listener").

📘 Notas de la clase

  • No es necesario escribir código en esta sección, se usarán ejemplos que ya vienen con ROS2.

  • Objetivo: validar que la instalación de ROS2 funciona correctamente.

  • Abrir dos terminales.

En la primera terminal:

  • Ejecutar el publicador (talker) con el siguiente comando:
    ros2 run demo_nodes_cpp talker
    (Nota: se puede usar Tab para autocompletar comandos)

  • Deberías ver líneas como:
    Publishing: 'Hello World: [número]'

En la segunda terminal:

  • Ejecutar el suscriptor (listener) con:
    ros2 run demo_nodes_cpp listener

  • Aparecerán mensajes como:
    I heard: 'Hello World: [número]'

Esto indica que el nodo "talker" está publicando mensajes y el nodo "listener" los está recibiendo.

Importante:

  • Si no ves los mensajes o aparece un error, probablemente no has "sourceado" bien el entorno de ROS2. Vuelve a la clase anterior y revisa los pasos de instalación.

  • Puedes detener los programas con Ctrl + C en cada terminal.

  • Si el "talker" se detiene, el "listener" dejará de recibir mensajes.

💾 Código trabajado / recursos (opcional)

  • Este ejercicio utiliza ejemplos ya incluidos en la instalación de ROS2. No es necesario descargar código adicional.
