2.2. Herramientas para programación adicionales

En esta clase aprenderás qué herramientas e IDEs se utilizan para trabajar con ROS 2 en Ubuntu, incluyendo cómo instalar y configurar Terminator, VS Code y Gedit para facilitar el desarrollo.

📘 Notas de la clase

Terminal y Terminator

  • Aunque puedes usar la terminal predeterminada de Ubuntu, se recomienda Terminator por su capacidad para dividir ventanas y trabajar con múltiples terminales a la vez.

  • Instalación de Terminator:
    sudo apt install terminator

Atajos útiles en Terminator:

  • Dividir terminal horizontalmente: Ctrl + Shift + O

  • Dividir terminal verticalmente: Ctrl + Shift + E

  • Maximizar/minimizar una terminal: Ctrl + Shift + X

  • Cerrar una terminal: Ctrl + Shift + W

  • Puedes encontrar más atajos buscando "Terminator keyboard shortcuts" en internet.

Editor de código: Visual Studio Code (VS Code)

  • VS Code ofrece una excelente integración con ROS y soporte para C++ y Python.

  • Instalación con snap:
    sudo snap install code --classic

⚠️ Extensión ROS: Cambios Importantes

La extensión original ROS de Microsoft ha sido deprecada y archivada el 2 de septiembre de 2025. Ha sido reemplazada por nuevas extensiones de Robot Developer Extensions.

Para ROS 2, instala:

Ranch-Hand-Robotics.rde-ros-2

Más información en: https://github.com/ms-iot/vscode-ros/issues/1306

Extensiones recomendadas:

  1. Robot Developer Extension for ROS 2 (Ranch-Hand-Robotics.rde-ros-2): extensión oficial actualizada y mantenida activamente para ROS 2.

  2. C/C++: proporciona autocompletado y resaltado de sintaxis.

  3. Python: soporte para scripts y nodos en Python.

  4. CMake Tools (by twx): para resaltar sintaxis en archivos CMakeLists.txt.

Puedes buscarlas e instalarlas desde el panel de extensiones (icono de bloques en la barra lateral izquierda de VS Code).

Editor de texto simple: Gedit

  • Para editar archivos sencillos se usará Gedit, aunque puedes usar alternativas como Nano o Vim.

  • Instalación (si no está instalada):
    sudo apt install gedit

📌 Resumen de herramientas instaladas

  • Terminator: terminal avanzada con soporte para múltiples paneles.

  • VS Code: editor principal con extensiones ROS, C++, Python y CMake.

  • Gedit: editor de texto simple para archivos rápidos.
