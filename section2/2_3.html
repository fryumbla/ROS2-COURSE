2.3. Instala ROS 2 Jazzy en Ubuntu 24.04

En esta clase instalarás ROS 2 Jazzy sobre Ubuntu 24.04, ya sea en un sistema nativo o dentro de una máquina virtual. Seguirás una serie de pasos recomendados para asegurarte de que el entorno quede correctamente configurado.

📘 Notas de la clase

Verifica la configuración del sistema

  • Abre una terminal y verifica la codificación:

    • Comando: locale

    • Debe mostrar codificación en UTF-8.

  • Si no aparece UTF-8, ejecuta los siguientes comandos para configurarla:

    • sudo locale-gen en_US en_US.UTF-8

    • sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8

    • export LANG=en_US.UTF-8

Agregar repositorios ROS 2 a tu sistema

  1. Añadir el repositorio universe:

    • sudo apt install software-properties-common

    • sudo add-apt-repository universe

  2. Agregar la clave GPG de ROS 2:

    • sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg

  3. Añadir el repositorio de ROS 2:

    • echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

Actualización del sistema

  • Ejecuta:

    • sudo apt update

    • Verifica que aparezca packages.ros.org entre los repositorios.

    • sudo apt upgrade

Herramientas de desarrollo ROS (opcional pero recomendado)

  • Instalar herramientas para desarrollo:

    • sudo apt install ros-dev-tools

Instalación de ROS 2 Jazzy

  • Hay dos opciones principales:

    • ros-jazzy-ros-base: instalación mínima, sin herramientas gráficas.

    • ros-jazzy-desktop: instalación completa (recomendada para el curso), incluye herramientas gráficas, demos y tutoriales.

  • Comando recomendado:

    • sudo apt install ros-jazzy-desktop

Nota: este proceso puede tomar entre 5 y 10 minutos (o más) según tu conexión y rendimiento del sistema.

Limpieza y mantenimiento post-instalación

  • Para mantener el sistema limpio y actualizado:

    • sudo apt update

    • sudo apt upgrade

    • sudo apt autoremove (para eliminar paquetes innecesarios)

    • También puedes actualizar paquetes individuales si es necesario.

📌 Verifica instalación

  • Ejecuta sudo apt upgrade nuevamente para confirmar que no quedan paquetes pendientes.
