6.3. Creación de un servidor de servicio en ROS 2 con C++

Aprenderás a crear un servidor de servicio en ROS 2 utilizando C++, proporcionando una interfaz de alto rendimiento para que otros nodos soliciten servicios.

📘 Notas de la clase

🔹 Revisión de la interfaz utilizada

  • Comando para ver la interfaz:
    ros2 interface show example_interfaces/srv/AddTwoInts

  • Contenido:

    • Request: dos campos enteros a y b (int64)

    • Response: un campo sum (int64)

  • Los --- indican la separación entre la solicitud y la respuesta.

🔹 Estructura del código en C++

  1. Crear archivo: add_two_ints_server.cpp

  2. Importar la interfaz:

    • #include "example_interfaces/srv/add_two_ints.hpp"

  3. Declarar el servidor como atributo privado del nodo:

    • Tipo: rclcpp::Service<example_interfaces::srv::AddTwoInts>::SharedPtr

    • Nombre sugerido: server_

🔹 Creación del servidor en el constructor

server_ = this->create_service<example_interfaces::srv::AddTwoInts>(
  "add_two_ints",
  std::bind(&AddTwoIntsServerNode::callback_add_two_ints, this, _1, _2)
);

  • Se define el tipo de servicio, el nombre del servicio, y el callback a ejecutar.

  • Se usa std::bind con _1 y _2 como placeholders de argumentos.

  • Para simplificar el uso de placeholders, se incluye:

    • using namespace std::placeholders;

🔹 Callback del servicio

void callback_add_two_ints(
  const std::shared_ptr<example_interfaces::srv::AddTwoInts::Request> request,
  const std::shared_ptr<example_interfaces::srv::AddTwoInts::Response> response)

  • Calcula la suma:

    • response->sum = request->a + request->b;

  • Muestra un mensaje informativo con los datos recibidos y calculados:

    • "a + b = sum"

🔹 Configuración del proyecto

  1. En CMakeLists.txt:

    • Añadir el ejecutable:

      • add_executable(add_two_ints_server src/add_two_ints_server.cpp)

    • Declarar dependencias:

      • rclcpp y example_interfaces

    • Añadir al install:

      • install(TARGETS add_two_ints_server ...)

  2. Compilar el paquete:

    • Utilizar colcon build --packages-select my_cpp_pkg

🔹 Validación del servicio

  • Ejecutar el nodo servidor:

    • ros2 run my_cpp_pkg add_two_ints_server

  • Verificar con:

    • ros2 node list

    • ros2 node info [nombre del nodo]

  • Probar con cliente Python:

    • ros2 run my_py_pkg add_two_ints_client

🔹 Compatibilidad multi-lenguaje

  • Puedes tener un cliente en Python y un servidor en C++ (o viceversa), y ambos se comunicarán correctamente si usan la misma interfaz.
