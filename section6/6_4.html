6.4. Creación de un cliente de servicio en ROS 2 con C++ con OOP

Aprenderás a crear un cliente de servicio en ROS 2 utilizando C++ con programación orientada a objetos (OOP), permitiendo una arquitectura robusta y mantenible para solicitar servicios.

📘 Notas de la clase

🔹 Estructura general del cliente

  1. Crear el archivo add_two_ints_client.cpp

  2. Incluir la interfaz:
    #include "example_interfaces/srv/add_two_ints.hpp"

  3. Crear una clase que herede de rclcpp::Node, por ejemplo: AddTwoIntsClientNode

🔹 Declaración del cliente

  • Declarar como atributo privado:
    rclcpp::Client<example_interfaces::srv::AddTwoInts>::SharedPtr client_;

  • Inicializar en el constructor:
    client_ = this->create_client<example_interfaces::srv::AddTwoInts>("add_two_ints");

🔹 Método para enviar solicitudes

Nombre sugerido: call_add_two_ints(int a, int b)

Pasos:

  1. Esperar a que el servicio esté disponible:
    Utiliza un bucle con client_->wait_for_service(1s)
    Imprime un mensaje cada segundo hasta que el servidor esté disponible.
    Usa using namespace std::chrono_literals; para los literales de tiempo.

  2. Crear y llenar el objeto de solicitud:
    auto request = std::make_shared<example_interfaces::srv::AddTwoInts::Request>();
    request->a = a;
    request->b = b;

  3. Enviar la solicitud de forma asíncrona y registrar un callback:
    client_->async_send_request(request, std::bind(&AddTwoIntsClientNode::callback_call_add_two_ints, this, std::placeholders::_1));

🔹 Callback para manejar la respuesta

Nombre sugerido: callback_call_add_two_ints

Firma:
void callback_call_add_two_ints(rclcpp::Client<example_interfaces::srv::AddTwoInts>::SharedFuture future)

Dentro del callback:

  • Obtener la respuesta:
    auto response = future.get();

  • Imprimir resultado:
    RCLCPP_INFO(this->get_logger(), "Result: %d", static_cast<int>(response->sum));

🔹 Llamada desde el main()

  • Crear el nodo:
    auto node = std::make_shared<AddTwoIntsClientNode>();

  • Enviar solicitudes:
    node->call_add_two_ints(10, 5);
    node->call_add_two_ints(12, 7);
    node->call_add_two_ints(1, 2);

  • Ejecutar el nodo:
    rclcpp::spin(node);

🔹 Configuración en CMakeLists.txt

  1. Agregar el ejecutable:
    add_executable(add_two_ints_client src/add_two_ints_client.cpp)

  2. Añadir dependencias:
    ament_target_dependencies(add_two_ints_client rclcpp example_interfaces)

  3. Instalar el ejecutable:
    install(TARGETS add_two_ints_client ...)

🔹 Compilación y ejecución

  1. Compilar:
    colcon build --packages-select my_cpp_pkg

  2. Ejecutar el servidor:
    ros2 run my_cpp_pkg add_two_ints_server

  3. Ejecutar el cliente:
    ros2 run my_cpp_pkg add_two_ints_client

🔹 Consideraciones

  • El cliente puede enviar múltiples solicitudes al mismo servidor.

  • El nodo debe estar en ejecución (rclcpp::spin) para recibir las respuestas.

  • Puedes integrar este cliente en nodos más complejos con otros publicadores, suscriptores, etc.

  • Si necesitas acceder a los datos del request en el callback, puedes almacenarlos como atributos del nodo.
