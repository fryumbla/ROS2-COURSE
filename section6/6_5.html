6.5. Introspección de servicios en ROS 2 con la línea de comandos

Aprenderás a usar herramientas de línea de comandos en ROS 2 para inspeccionar, listar y analizar servicios disponibles en tu sistema.

📘 Notas de la clase

🔹 Comandos generales para trabajar con servicios

  • Ver todos los subcomandos disponibles:
    ros2 service -h

  • Ver los servicios disponibles:
    ros2 service list

  • Ver los nodos disponibles:
    ros2 node list

  • Obtener información de un nodo (incluye los servicios que expone):
    ros2 node info [nombre_del_nodo]

🔹 Servicios generados automáticamente

  • Cada nodo en ROS 2 crea automáticamente varios servicios relacionados con parámetros:

    • Ejemplos: /node_name/describe_parameters, /node_name/get_parameter_types

  • Si tú no los creaste, son añadidos por ROS 2 para gestionar los parámetros del nodo.

🔹 Ver el tipo de un servicio

  • Obtener el tipo de un servicio específico:
    ros2 service type /add_two_ints

  • Ver el contenido de la interfaz:
    ros2 interface show example_interfaces/srv/AddTwoInts
    Muestra los campos de solicitud (request) y respuesta (response), separados por ---.

🔹 Llamar a un servicio desde la terminal

Comando general:
ros2 service call [nombre_del_servicio] [tipo_de_interfaz] "[request]"

Ejemplo:
ros2 service call /add_two_ints example_interfaces/srv/AddTwoInts "{a: 3, b: 7}"

  • Asegúrate de:

    • Usar comillas dobles " para encapsular la solicitud

    • Usar llaves {} y colocar un espacio después de los : para evitar errores de sintaxis

También puedes omitir campos del request (si son tipos primitivos), y ROS 2 asignará valores por defecto (por ejemplo, 0 para enteros, cadena vacía para strings).

🔹 Llamar a un servicio con RQt

  1. Iniciar RQt:
    rqt

  2. Ir a Plugins > Services > Service Caller

  3. Seleccionar el servicio, por ejemplo /add_two_ints

  4. Ingresar los valores de los campos a y b

  5. Presionar Call para enviar la solicitud

Ventajas:

  • No necesitas recordar la sintaxis del terminal

  • Ideal para servicios con pocos y simples parámetros

  • Muestra el log en el servidor y la respuesta recibida

🔹 Alternativas para probar servicios

  • Desde la terminal con ros2 service call

  • Desde RQt con el plugin Service Caller

  • Desde scripts en Python o C++ (como los creados en clases anteriores)

💾 Código trabajado / recursos

  • Interfaz de ejemplo utilizada: example_interfaces/srv/AddTwoInts

  • Servidores compatibles: add_two_ints_server.py (Python) y add_two_ints_server.cpp (C++)

  • Clientes compatibles: add_two_ints_client.py, add_two_ints_client.cpp
