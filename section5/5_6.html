5.6. Introspección de tópicos en ROS 2 usando la línea de comandos

En esta clase aprenderás a explorar y analizar los tópicos disponibles en tu sistema ROS 2 utilizando herramientas de línea de comandos como ros2 topic list, ros2 topic echo, ros2 topic info y ros2 topic type.

📘 Notas de la clase

🔍 Comandos básicos para explorar tópicos

  • Listar tópicos activos:

    • ros2 topic list

    • Muestra todos los tópicos disponibles en el sistema.

    • Solo aparecen los tópicos que tienen al menos un publicador o suscriptor activo.

  • Obtener información detallada de un tópico:

    • ros2 topic info /robot_news

    • Muestra:

      • Tipo de mensaje usado (ej. example_interfaces/msg/String)

      • Número de publicadores y suscriptores actuales

  • Visualizar mensajes en tiempo real:

    • ros2 topic echo /robot_news

    • Se convierte en un suscriptor temporal y muestra los mensajes publicados en el tópico

📦 Inspeccionar tipos de mensajes

  • Ver la estructura de un tipo de mensaje:

    • ros2 interface show example_interfaces/msg/String

    • Útil para confirmar los campos disponibles en un tipo de mensaje

    • En este caso, solo muestra un campo: string data

📈 Medir frecuencia y ancho de banda

  • Frecuencia de publicación (Hz):

    • ros2 topic hz /robot_news

    • Muestra la tasa promedio de mensajes por segundo (ej. 2 Hz = un mensaje cada 0.5 s)

  • Consumo de ancho de banda:

    • ros2 topic bw /robot_news

    • Muestra el volumen de datos transmitido por segundo (en bytes/s)

🧪 Publicar desde la terminal

  • Comando para publicar mensajes manualmente:

    • ros2 topic pub -r 5 /robot_news example_interfaces/msg/String "{data: 'Hello from the terminal'}"

    • Explicación:

      • -r 5 → frecuencia: 5 mensajes por segundo

      • /robot_news → nombre del tópico

      • example_interfaces/msg/String → tipo de mensaje

      • "{data: 'Hello from the terminal'}" → contenido del mensaje

    • Solo funciona con mensajes simples (ej. tipo String). No es viable para estructuras complejas.

  • Mientras este comando se ejecuta, puedes usar ros2 topic echo /robot_news para ver los mensajes publicados.

🧠 Complemento: Comandos ros2 node

  • Ver nodos activos:

    • ros2 node list

  • Ver información de un nodo específico:

    • ros2 node info /robot_news_station

    • Muestra los tópicos que publica y suscribe, así como los tipos de mensajes utilizados
