10.1. Creación de un archivo URDF y visualización con RViz

Aprenderás a crear tu primer archivo URDF, entender su estructura XML básica y visualizar tu robot en RViz2.

📘 Notas de la clase

📂 1. Creación del archivo URDF

  • No es necesario crear un paquete de ROS todavía. El enfoque es entender la estructura del archivo URDF.

  • Crear un archivo en tu directorio personal (por ejemplo: ~/my_robot.urdf).

  • Editarlo con Visual Studio Code u otro editor de tu preferencia.

Inicio del archivo URDF:

  • Aunque la extensión es .urdf, el contenido es XML.

  • Añadir la cabecera XML:

    < ?xml version="1.0" ? >

  • Declarar el robot:

    <robot name="my_robot"> ... </robot>

🧱 2. Añadir el primer link

  • Un link representa una parte rígida del robot.

  • Se suele comenzar con base_link porque algunas aplicaciones (como navegación) lo requieren.

Ejemplo de estructura para el primer link:

<link name="base_link"> ... </link>

🧩 3. Visualización del link

  • Añadir una etiqueta visual dentro del link para representar su apariencia.

  • Dentro de visual, incluir:

    • geometry con un box y su tamaño.

    • origin para posicionamiento y orientación.

Ejemplo:

<visual>
<geometry>
<box size="0.6 0.4 0.2"/>
</geometry>
<origin xyz="0 0 0" rpy="0 0 0"/>
</visual>

  • Medidas en metros. Si usas sistema imperial, deberás convertir.

🧭 4. Posicionamiento del visual con origin

  • Por defecto, el centro del box se ubica en el origen.

  • Para que la base del box esté sobre el suelo (y no atravesándolo), desplazar en z la mitad de la altura (0.1 m si el alto es 0.2 m):

    <origin xyz="0 0 0.1" rpy="0 0 0"/>

👁️ 5. Visualización con RViz

  • Asegúrate de tener instalado el paquete urdf_tutorial.

Comando:

ros2 launch urdf_tutorial display.launch.py model:=my_robot.urdf

  • Esto abrirá RViz y mostrará el base_link como un cubo centrado en el origen.

💾 Código trabajado / recursos

Estructura mínima del archivo my_robot.urdf:

< ?xml version="1.0" ? >
<robot name="my_robot">
<link name="base_link">
<visual>
<geometry>
<box size="0.6 0.4 0.2"/>
</geometry>
<origin xyz="0 0 0.1" rpy="0 0 0"/>
</visual>
</link>
</robot>
