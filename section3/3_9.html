3.9. OOP Template

En esta clase aprenderás sobre la plantilla de programación orientada a objetos para estructurar de manera profesional y escalable tus nodos en ROS 2.

Aquí tienes una plantilla que puedes usar para todos tus nodos. Asegúrate de entender qué hace cada línea revisando (si es necesario) las clases anteriores.

OOP Python Code Template for Nodes

    #!/usr/bin/env python3
    import rclpy
    from rclpy.node import Node
     
     
    class MyCustomNode(Node): # MODIFY NAME
        def __init__(self):
            super().__init__("node_name") # MODIFY NAME
     
     
    def main(args=None):
        rclpy.init(args=args)
        node = MyCustomNode() # MODIFY NAME
        rclpy.spin(node)
        rclpy.shutdown()
     
     
    if __name__ == "__main__":
        main()

    class MyCustomNode : public rclcpp::Node // MODIFY NAME

        }
     
    private:
    };
     
    int main(int argc, char **argv)
    {
        rclcpp::init(argc, argv);
